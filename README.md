# Customer Segmentation using K-Means

## Overview
This project is a Streamlit web app that performs customer segmentation using K-Means clustering.

## Features
- Upload your own dataset
- Visualize clusters
- Elbow method for optimal clusters
- Cluster summary

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## Run the app

streamlit run app.py
